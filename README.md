# XOR inference with cuBLAS/cuDNN

## Overview

The goal of this learning project is to implement a simple neural network.  
Using the libraries cuBLAS and cuDNN, the network will perform an XOR on a set of data.  
The Python program is used to train the neural network and store the weights.  
The C++ program loads the weights and performs the inference.  
The model is made up of two linear layers with sigmoid activation functions.

## Prerequisites
[python3](https://www.python.org/downloads)  
[numpy](https://numpy.org)  
[pytorch](https://pytorch.org)  
[CUDA Toolkit 12.6](https://developer.nvidia.com/cuda-downloads)  
[cuDNN 9.5.0](https://developer.nvidia.com/cudnn-downloads)  

## Build and Run

run train.py to build the neural network weights
```bash
python3 train.py
```

### Windows
The Windows program is built using the Visual Studio 2022 IDE.

### Linux
The Linux program is built using makefiles. To use the makefiles, just run make:
```
$ make
```

After building the project, you can run the program using the following command:

```bash
make run
```
